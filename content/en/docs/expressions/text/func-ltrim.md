## Syntax



```
func.ltrim(value_string, string_to_trim)
```


## Examples



```
func.ltrim('texttotrimplaidcloud', 'texttotrim') --> plaidcloud  
func.ltrim('      plaidcloud')                   --> plaidcloud
```
