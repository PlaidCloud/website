## Syntax



```
func.rtrim(value_string, string_to_trim)
```


## Examples



```
func.rtrim('Plaidcloud    ') --> Plaidcloud  
func.rtrim('PlaidcloudXXXXX','XXXXX') --> Plaidcloud
```
